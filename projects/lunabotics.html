Autonomous Lunar Rover

Northwestern Space Technology And Rocketry Society | Software Lead

Rover Image Here

Overview

Developing autonomous navigation and control systems for NASA's Lunabotics competition. The rover must autonomously navigate simulated lunar terrain, excavate regolith, and deposit it in designated zones - all without human intervention.

Technical Challenges

  • Autonomous Navigation: Implementing SLAM (Simultaneous Localization and Mapping) for GPS-denied environments using LiDAR and camera fusion
  • Obstacle Detection: Real-time terrain analysis to avoid rocks and craters
  • Path Planning: Dynamic trajectory generation using ROS2 Nav2 stack
  • Sensor Integration: Fusing data from multiple sensors (IMU, encoders, cameras) for robust state estimation

My Contributions

As Software Lead, I:

  • Architected the ROS2 navigation stack with custom behavior trees
  • Developed computer vision pipeline for autonomous excavation zone detection
  • Implemented sensor fusion algorithms for localization accuracy within 10cm
  • Integrated motor controllers (Spark Max) with CAN bus communication
  • Set up Gazebo simulation environment for testing before hardware deployment

Technologies Used

ROS2 Jazzy Python C++ OpenCV Gazebo Nav2 LiDAR CAN Bus
