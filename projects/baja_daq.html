Baja SAE Data Acquisition System

Motorcats Baja SAE | Electronics Team

DAQ Dashboard Interface

Overview

Developed a comprehensive data acquisition and telemetry system for the Motorcats Baja SAE competition vehicle. The system monitors critical vehicle parameters in real-time, enabling data-driven optimization of suspension tuning, powertrain performance, and driver feedback during testing and competition events.

System Requirements

  • Data Logging: Continuous recording at 100Hz for post-event analysis of suspension travel, wheel speeds.
  • Wireless Telemetry: Live data streaming to pit crew during competition runs
  • Rugged Design: waterproofing and vibration resistance for off-road conditions
  • Low Power: 8+ hour battery life for full competition day without recharging

Hardware Architecture

Built around a ESP32 microcontroller for sensor data collection, with a CAN transceiver for communication with engine ECU and powertrain controllers. Custom PCB design includes signal conditioning circuits for analog sensors, voltage regulators for stable 5V/3.3V power rails, and automotive-grade connectors for reliability.

Sensor suite includes: wheel speed Hall sensors (4x).

Software Development

Implemented embedded firmware in C/C++ using interrupt-driven architecture for deterministic real-time performance.

Telemetry system uses 915MHz LoRa radio for long-range wireless communication to pit station.

Key Technical Challenges

  • Noise Immunity: Electrical noise from ignition system and motors
  • Vibration Hardening: needs to survive 6+ hours of off-road abuse
  • Power Management: Designed smart power sequencing to protect microcontroller during engine cranking voltage sags
  • Data Synchronization: Implemented GPS time-stamping for precise correlation of sensor data with video footage and lap timing

Technologies Used

ESP32 C/C++ CAN Bus PCB Design Real-Time Systems Signal Processing

Data Analysis & Insights

Post-processing pipeline analyzes logged data to extract key performance metrics. Developed automated scripts to calculate linear velocity, identify wheel slip events, and correlate engine RPM with vehicle speed to optimize eCVT tuning.
